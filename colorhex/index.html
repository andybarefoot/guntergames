Color Hex

About Color Hex

COLOR HEX: The tricky puzzle game where you must color the hexagons to create the correct pattern.
As the patterns get more complicated the puzzle becomes more difficult.

The game has two play modes:

In Mix Mode when two colors collide they mix to create a new shade. With each color affecting several hexagons, and each hexagon being a mixture of up to 3 colors, place your colors precisely if you are to match the pattern exactly.

In Swap Mode placing a new color replaces the old color, but with colors coming at each hexagon from 3 directions you will need to place he colors in exactly the right order.

Complete each level to unlock the next. Complete them in the minimum number of moves for a perfect score.
